Overview

Use this workflow to join a team with an invite link or an 8-character invite code.

You will redeem the invite, then confirm you landed in the correct dashboard for your role.

In This Article

  • Choose the right invite path.
  • Join with a link or manual code.
  • Complete sign-in or account creation.
  • Confirm your dashboard and team access.
  • Fix common invite and sign-in issues.

Who Is This For

  • Parent users joining a child’s team.
  • Coach users joining team staff access.
  • Admin users joining admin/team access.

Prerequisites

  • You have a valid invite link or 8-character invite code.
  • You are ready to sign in or create an account.
  • You can use the same email address that received the invite.
  • You can stay in the invite flow until you see success.

Choose Your Path

  1. I have an invite link and I am already signed in.

Open the link and continue through invite confirmation.

  1. I have an invite link and I am not signed in.

Open the link, then sign in or create an account when prompted.

  1. I only have an 8-character code.

Open invite acceptance, choose manual code entry, then enter the code.

  1. I am a parent adding another player after I already joined.

Use Redeem Code in Parent Dashboard with the new code.

Step-by-Step Workflow

  1. Open your invite link.

If the page shows Processing Invite..., wait for it to finish.

  1. Confirm your email if prompted.

If you opened the link on another device, enter the invited email and select Continue.

  1. Enter your invite code if needed.

If manual entry appears, type the 8-character code and select Join Team.

  1. Sign in or create your account.

Existing account: sign in with email/password or Google.

New account: select Sign Up, complete the form, and provide the activation code if asked.

  1. Complete invite redemption.

Return to the invite flow if prompted. Wait for You’re In! and the automatic redirect.

  1. Verify your role-based destination.

Parent: confirm Parent Dashboard shows the player/team.

Coach/Admin: confirm My Teams shows the new team card.

  1. Open the team once.

Select View (and optionally Chat) to confirm access is active.

Common Questions

  • Do I need to use the same email as the invite?

Yes. Use the invited email when asked to confirm your email.

  • Can I join with only a code?

Yes. Use manual code entry, then complete sign-in.

  • I created an account but I am not on a team yet. What should I do?

Reopen the invite link (or re-enter the code) and complete invite redemption.

  • Where should I land after joining?

Parent users should land in Parent Dashboard. Coach/Admin users should land in My Teams.

Recovery & Troubleshooting

  • Invalid or expired code

Re-enter the code exactly (8 characters). If it still fails, ask for a fresh invite.

  • Invalid email or expired link

Use the invited email address and reopen the latest invite link.

  • Sent back to Login repeatedly

Finish sign-in first, then reopen the invite link or retry manual code entry.

  • Joined, but no team appears

Refresh the dashboard, then sign out and sign back in once.

Parent users: check Parent Dashboard.

Coach/Admin users: check My Teams.

  • Parent cannot see newly added player

Use Redeem Code in Parent Dashboard for that player’s code.

If still missing, ask the coach/admin to resend the parent invite.
