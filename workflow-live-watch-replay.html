Overview

Use this workflow to open a game, follow it live, and switch to replay or the match report after the game ends.

You can start from a shared watch link, the home page, the team page, or the match report page.

In This Article

  • Where to open a live or completed game.
  • How to follow score, stream, plays, stats, chat, and reactions during live action.
  • How to move to replay and match report after the final score.
  • What to do when live, replay, chat, or sharing is not working.

Who Is This For

  • Parent: Watch live score and stream, join chat, and watch replay later.
  • Member: Follow live updates, then review replay or match report.
  • Coach: Watch live, share game and report links, and direct families to replay/report.
  • Admin: Same as Coach, plus full-access match report editing (for example, summary updates).

Prerequisites

  • You have a watch link, or you can open the game from the home page or team page.
  • The game is scheduled, live, or completed.
  • You have a stable internet connection.
  • For team-level sharing or report actions, sign in with your team account.

Choose Your Path

  1. Shared link path

Open the shared watch link directly.

  1. Home page path (index.html)

Open Live & Upcoming Games and select Watch Now or View Details.

Open Recent Replays and select Watch Replay.

  1. Team page path (team.html)

Find the game card.

For active or upcoming games, select Live Now or View Live.

For completed games, select Replay or View Report.

  1. Match report path (game.html)

Open the game report and select Watch Replay or Live Now (when shown).

Step-by-Step Workflow

  1. Open the game in live view.

Go to live-game.html. Confirm the scoreboard is visible at the top.

  1. Check game status.

If the game is live, look for the LIVE badge and current score.

If the game has not started, you may see Game Not Live Yet.

If the game is finished, use Watch Replay from the end-state prompt.

  1. Follow the game feed.

Use tabs for Plays, Stats, and Chat.

Open Video when a stream is available.

Track score, period, clock, and viewer count in the header.

  1. Interact during the game (optional).

Send a message in Chat.

Set or update your display name if prompted.

Use the reaction bar (fire, clap, wow, heart, hundred).

  1. Move to post-game content.

Select Watch Replay to enter replay mode.

Use replay controls: play/pause, scrubber, and speed (1x to 50x).

Select View Game Details or Match Report to open the full report page.

  1. Share the game or report.

Parent/Member: Copy the page URL and share it.

Coach/Admin: Use built-in Share on live/team pages and Share Report on report/team pages.

Common Questions

  • How do I confirm the game is live?

Look for the LIVE badge and pulsing live indicator on the live page.

  • Why is replay not available yet?

Replay appears after live tracking is completed for that game.

  • Can I open the page before tip-off?

Yes. Open the game page and wait there. Status updates on that screen.

  • Where is the match report?

Use Match Report on the live page or View Report on team/game pages.

  • Why is the Video tab missing?

Some games do not have an active embedded stream. Use plays, stats, and chat instead.

Recovery & Troubleshooting

  • Game link opens but game does not load.

Confirm the URL includes both teamId and gameId.

Ask a coach/admin to resend the link from Share.

  • Game says “not live yet” but should be live.

Confirm the scheduled time.

Refresh after one minute and check again.

  • Replay button is missing after final score.

Open team.html and select View Report first.

Then return and retry Watch Replay.

  • Chat message will not send.

Check for a pregame chat lock notice.

If prompted, enter a display name and resend.

  • Share sheet does not open.

Copy the page URL manually and send it in your group chat.
