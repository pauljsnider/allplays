Overview

Use this workflow after a game ends to verify results, clean up postgame details, and share links.

Start in team.html for fast sharing. Use game.html for a full review and edits.

In This Article

  • Pick the fastest path for your postgame goal.
  • Verify final score, events, and player performance.
  • Edit summary and score sheet if you have full access.
  • Share report and replay links.
  • Fix common postgame issues quickly.
  • Check replay availability state before sharing.
  • Browse the saved highlight library for the game.
  • Embed or share the team scoreboard widget.

Who Is This For

  • Parent: Review results, replay, and player details.
  • Coach: Finalize report quality, verify stats, and publish results.
  • Admin: Support edits, accuracy checks, and sharing.
  • Member: Open shared report and replay links.

Prerequisites

  • The game is completed and game data exists.
  • You have the correct team or game link.
  • To edit summary or score sheet, sign in with full team access (Coach/Admin).

Choose Your Path

  • Quick share: Open team.html and share from Recent Results.
  • Full review: Open game.html to confirm summary, play-by-play, and player performance.
  • Player follow-up: From game.html, open a player in player.html.
  • Replay sharing: Share replay from team.html, game.html, or live-game.html when available.
  • Check replay status before sharing: open game.html to see the replay state (unavailable, processing, failed, or ready).
  • Browse game highlights: open the game media hub to see saved highlight clips with timestamps and player tags.

Step-by-Step Workflow

  1. Open the completed game report.

In team.html, go to Recent Results, then select View Report.

  1. Confirm final game info.

Verify opponent, date, final score, and outcome label.

  1. Review report sections.

Check Match Summary, Play-by-Play, and Player Performance.

  1. Edit summary if needed (Coach/Admin only).

Select Edit Summary, update content, then select Save Summary.

  1. Optionally generate summary text (Coach/Admin only).

Select Generate AI Summary, review it, edit as needed, and save.

  1. Update score sheet if needed (Coach/Admin only).

Use Upload Score Sheet, preview, and save.

  1. Correct score sheet if needed (Coach/Admin only).

Use Remove Score Sheet, upload the correct image, and save.

  1. Review player-level detail.

Open a player from Player Performance to view Game Stats, Season Averages, and Game Events in player.html.

  1. Share final links.

Use Share Report from game.html or team.html. Add replay links when available.

  1. Validate the shared experience.

Open the shared link once to confirm report load and replay playback.

Replay Availability States

Before sharing a replay link, check the game page for the current replay state:

  • Ready: Replay is available. Share the link freely.
  • Processing: Replay is being prepared. Playback is blocked. Check back in a few minutes.
  • Failed: Replay processing failed. Playback is blocked. Contact support if this persists.
  • Unavailable: No replay exists for this game. The video area shows a clear unavailable state rather than an empty player.

Saved Highlight Library

Coaches and videographers can save replay highlights during or after a game. Highlights are accessible from the game media hub.

  • Each highlight shows title, timestamp, duration, player tags, and play context.
  • Click a highlight to seek the replay to that moment (when replay is available).
  • Use highlights to quickly navigate key plays for coaching review or sharing.

Embeddable Scoreboard Widget

Coaches and admins can share a compact, read-only team scoreboard for embedding on external websites.

  1. Open the team page and locate the Scoreboard Widget option.
  2. Copy the iframe embed snippet to embed the scoreboard on an external website.
  3. Or copy the direct widget link to share as a standalone URL.

Common Questions

  • Why can’t I edit the summary or score sheet?

You need full team access (Coach/Admin).

  • Can parents or members edit postgame content?

No. They can review and share links only.

  • Why is there no report for this event?

Practice events do not have game reports.

  • Where can I share replay only?

Share from completed live games in team.html, game.html, or live-game.html.

  • How do I review one player after the game?

Open that player from Player Performance and review Game Stats and Game Events.

  • Replay link I shared isn't working for others

Check the replay state on the game page first. If it shows Processing or Failed, the replay is not yet accessible. Share the link again once the state shows Ready.

  • Where are saved highlights?

Open the game media hub on the game page. Highlights are listed with title, timestamp, player tags, and a seek button to jump to that moment in the replay.

Recovery & Troubleshooting

  • Summary is missing or outdated:

Coach/Admin should open Edit Summary, save again, refresh, and re-share.

  • Score sheet is incorrect:

Coach/Admin should remove it, upload the correct image, and verify the report.

  • Share action fails:

Copy the report URL from the browser and send it directly.

  • Replay link is missing:

Replay appears only after live game status is completed. Share report now and replay later.

  • Stats look incomplete:

Recheck play-by-play and player game events, then reopen the report.

  • Link opens the wrong place:

Return to team.html, open Recent Results, and launch the report again.

  • Replay shows Processing for a long time: Processing time varies. If it shows Failed after 30+ minutes, contact support.
  • Summary Save button unresponsive after first save: This is fixed — the Save Summary button now re-enables each time the editor opens. Reload the page if you still see the issue.
