Overview

Use this workflow to get into ALL PLAYS fast.

Choose the path that matches your situation: new account, sign-in, password reset, or email verification.

In This Article

  • Choose the right account access path.
  • Create a new account with an activation code.
  • Sign in with email/password or Google.
  • Verify your email now or later.
  • Reset your password.
  • Resolve common login and verification issues.

Who Is This For

  • Parent: Create an account from a code, or sign in to continue team access.
  • Coach: Create or sign in, then verify email and keep access active.
  • Admin: Create or sign in, then continue invite-based onboarding when needed.
  • Member: Create or sign in with an activation code and confirm access.

Prerequisites

  • You can access your email inbox now.
  • You have a stable internet connection and supported browser.
  • For first-time signup, you have an 8-character activation code.
  • If you prefer Google sign-in, you have a Google account.

Choose Your Path

  • Path A: I am new and have an activation code.
  • Path B: I already have an account.
  • Path C: I forgot my password.
  • Path D: I need to verify my email or resend verification.
  • Path E: I signed in with an email link and want to set a password.

Step-by-Step Workflow

  1. Open index.html and select Get Started Now, or go to login.html.
  2. Follow the path that matches your situation.

Path A: Create a new account

  1. On login.html, select Sign Up.
  2. Enter your email, password, and confirm password.
  3. Enter your 8-character activation code.
  4. Select Create Account.
  5. Optional: Select Continue with Google in Sign Up mode. You still need an activation code.
  6. After signup, go to verify-pending.html.
  7. Open your verification email and select the link.
  8. If needed, select Resend Verification Email.

Path B: Sign in to an existing account

  1. On login.html, stay in Login mode.
  2. Sign in with Sign In (email/password) or Continue with Google.
  3. If you are a Parent or Admin on an invite link, continue invite acceptance.
  4. If you are a Coach, Admin, or Member with normal sign-in, continue to your dashboard.
  5. If sign-in fails, go to Path C.

Path C: Reset a forgotten password

  1. On login.html, enter your account email.
  2. Select Forgot Password?.
  3. Open the reset email and select the link.
  4. On reset-password.html, enter and confirm a new password (minimum 6 characters).
  5. Return to login.html and sign in with the new password.

Path D: Verify email or resend verification

  1. On verify-pending.html, select Resend Verification Email if needed.
  2. You can select Continue to Dashboard while verification is pending.
  3. If already signed in, open profile.html.
  4. If you see Email not verified, select Resend Email.
  5. Complete verification from your inbox.

Path E: Set a password after email-link sign-in

  1. Open profile.html.
  2. If Set a Password appears, enter and confirm a password (6+ characters).
  3. Select Set Password.
  4. Use that password for future logins on login.html.

Common Questions

  • Do I need an activation code for my first account?

Yes. New account creation requires an 8-character activation code.

  • Can I sign up with Google instead of a password?

Yes. In Sign Up mode, a valid activation code is still required.

  • Do I have to verify my email before using the app?

No. You can continue to dashboard and verify later from profile.html.

  • Why is Forgot Password? not working?

Enter your email first, then select Forgot Password?.

Recovery & Troubleshooting

  • No verification or reset email arrived:

Check spam/junk, confirm your email address, then resend from verify-pending.html or profile.html.

  • Invalid or expired verification/reset link:

Start again from login.html and use the newest email.

  • Password reset fails:

Make sure both fields match and the password is at least 6 characters.

  • Too many resend attempts:

Wait a few minutes, then try again.

  • Google sign-in is blocked:

Allow pop-ups or redirects in your browser, then retry. If needed, use email/password.

  • No activation code available:

Ask your coach or admin for a new code or signup link.
