Overview

Use this workflow to keep families, coaches, and admins aligned before, during, and after events.

Start by choosing the path that matches your goal: team chat, RSVP, rideshare, or live-game chat.

In This Article

  • Pick the right path for messaging, availability, or rideshare.
  • Send clear team updates and ask @ALL PLAYS questions.
  • Set event status as Going, Maybe, or Can't Go.
  • Offer rides, request spots, and manage request decisions.
  • Recover quickly from common send, access, and RSVP issues.

Who Is This For

  • Parent: Share updates, set availability, and coordinate rides.
  • Coach: Post reminders, keep chat focused, and moderate chat.
  • Admin: Support communication standards and moderate chat.

Prerequisites

  • You are signed in.
  • You have access to at least one team.
  • You opened a team context from your dashboard, team view, calendar, or live game page.
  • For RSVP buttons, the event is already tracked in the team schedule.

Choose Your Path

  1. Send a team update or reminder now: use team chat.
  2. Submit event availability: use the schedule list or calendar event view.
  3. Coordinate transportation: use event rideshare.
  4. Communicate during a game: use live-game chat when enabled.

Step-by-Step Workflow

Path A: Team Messages and Moderation

  1. Open team chat from your current team context.
  2. Write one clear update per message.
  3. Optional: add voice input or attach photos or short videos (up to 5 MB each).
  4. To ask the assistant, include @ALL PLAYS and your question in the same message.
  5. Select Send.
  6. Confirm the message appears in the thread.
  7. Add reactions for quick acknowledgment.
  8. Edit or delete your own message if plans change.
  9. If you are a Coach or Admin, remove off-topic or inappropriate messages from others when needed.
  10. For live-game chat, post short updates. If chat is locked, retry at game time.

Path B: RSVP Availability

  1. Open the event from schedule list, calendar, or day detail.
  2. Confirm team and player context.
  3. Select Going, Maybe, or Can't Go.
  4. Confirm the selected state stays active.
  5. Verify the availability summary updates.

Path C: Family Logistics (Rideshare)

  1. Open the event rideshare section.
  2. To drive, select Offer Ride.
  3. Set seats and direction, add an optional note, then save.
  4. To request a seat, choose the child and select Request Spot.
  5. Cancel your request if plans change.
  6. If you own the offer, use Confirm, Waitlist, or Decline for each request.
  7. Close or reopen your offer as needed.

Common Questions

  • Where should I post updates? Use team chat for general updates and live-game chat for real-time game communication.
  • Why are RSVP buttons missing? Availability controls appear after the event is tracked in the team schedule.
  • Who can moderate chat? Everyone can manage their own messages. Coaches and admins can also remove messages from others.
  • How do I ask the assistant a question? Type @ALL PLAYS and your question in one message.

Recovery & Troubleshooting

  • Message failed to send: Check your connection, resend, and keep the message short.
  • Image upload failed: Retry with a valid image at or below 5 MB.
  • No response from @ALL PLAYS: Resend with a clear question after the mention.
  • Live-game chat is locked: Wait until game time, then post again.
  • RSVP did not save: Reopen the event, reselect your response, and confirm the button stays active.
  • Team chat access error: Return to team context and reopen chat from that team.
