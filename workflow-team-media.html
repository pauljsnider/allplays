Manage Team Media and Albums

Organize photos, videos, and highlights into albums, control visibility, and let members contribute uploads.

4 min read Updated from live product pages

On this page

Who can use this

Admin Coach Parent

Overview

Team Media is a team-scoped library where coaches and admins organize photos and video links into named albums. Members can browse member-visible albums and upload photos directly from the app. Admins have full creation, moderation, and bulk management controls — including visibility settings, cover selection, and bulk delete or move operations.

In This Article

  • Open Team Media and browse existing albums
  • Create and manage albums with visibility controls
  • Add YouTube and Vimeo video links to an album
  • Upload photos with per-file progress tracking
  • Set an album cover image
  • Use admin bulk operations: select, delete, move, and reorder
  • Delete individual items as an uploader or admin

Who Is This For

  • Admin — create and manage albums, set visibility, moderate uploads, set cover images, perform bulk operations (delete, move, reorder).
  • Coach — add content (photos and video links) to albums, manage folders.
  • Parent / Member — browse member-visible albums, view photos and video links, and upload photos when admin has enabled member upload access.

Prerequisites

  • Signed in with a valid account.
  • Team access (member, coach, or admin on the team).
  • For member photo uploads: admin must have enabled member upload access for the team.

Choose Your Path

  • Create or organize albums — admin uses the album management controls to add, rename, reorder, or delete albums.
  • Add video links — admin or coach pastes a YouTube or Vimeo URL into any album.
  • Upload photos — member or admin selects files from within an album view.
  • Browse the gallery — any member with team access opens Team Media to view visible albums.
  • Bulk manage items — admin uses multi-select to bulk delete, move to another album, or reorder items.

Step-by-Step Workflow

1. Open Team Media

Access Team Media from the team management banner. The Media entry point is available to admins and coaches. Members with team access can reach the gallery through their team view. The library opens to the album list.

2. Create and manage albums (Admin)

Select Create Album, enter a name, and choose a visibility setting:

  • Member-visible — all team members including parents can browse this album.
  • Manager-only — visible only to admins and coaches; hidden from parents and general members.

After creation, use the album management controls to rename, reorder, or delete any album. Deleting an album removes all items inside it.

3. Add video links

Open an album and select Add Video Link. Paste a valid YouTube or Vimeo URL. The link is saved as a typed media item and renders in the gallery with its metadata (title, thumbnail when available). Video links are stored against the album and persist across sessions.

4. Upload photos

Open an album and select Upload Photos. Choose one or more image files. Each file uploads independently — a per-file progress indicator is shown. If a single file fails (due to size or format), that failure is shown inline and does not block the remaining files from completing. Each uploaded photo is stored with its URL, storage path, uploader identity, file size, MIME type, and upload timestamp.

5. Set an album cover (Admin)

Inside an album, select any photo and choose Set as Cover. The selected photo becomes the cover image shown on the album card in the main gallery view. You can change the cover at any time by selecting a different photo.

6. Browse the gallery

Members see all member-visible albums in the Team Media gallery. Opening an album shows photo items with the uploader name, upload date, and a download link. Video link items display inline — click a video item to open the linked YouTube or Vimeo page.

7. Bulk operations (Admin)

Enter multi-select mode within an album. Select one or more items, then choose an action:

  • Bulk delete — removes all selected items. A confirmation prompt appears before any destructive action.
  • Move — transfers selected items to another album you choose from a list.
  • Reorder — drag selected items to a new position within the album.

8. Delete individual items

Any uploader can delete their own photos. Admins and team owners can delete any item regardless of who uploaded it. Select the item and choose Delete. Deletion is permanent and cannot be undone.

Common Questions

  • Can parents see all albums? No — only albums with member-visible visibility are shown to parents and general members. Manager-only albums are hidden.
  • Can members upload without admin approval? The upload control is available when an admin has enabled member upload access for the team. Individual uploads do not require per-upload approval.
  • Are video links stored in the database? Yes — each video link is saved as a typed media item attached to the album, with the URL and metadata stored in Firestore.
  • What happens to items when an album is deleted? All items inside the album — photos and video links — are permanently removed along with the album.

Recovery and Troubleshooting

  • Album not visible to members — open the album settings and confirm visibility is set to member-visible. Manager-only albums are always hidden from parents.
  • Photo upload failed — check the file size and format. Per-file failures are reported inline next to the affected file; other files in the same batch are not affected. Retry the failed file after reducing its size if needed.
  • Video link not rendering correctly — confirm the URL is a direct YouTube or Vimeo link (e.g., https://www.youtube.com/watch?v=... or https://vimeo.com/...). Other video platforms are not supported.
  • Upload controls not visible to members — an admin must enable member upload access in the team settings before the upload option appears for non-admin users.
